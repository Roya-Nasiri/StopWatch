# StopWatch 
# ⏱️ Stopwatch App

Demo : (https://roya-nasiri.github.io/StopWatch/)

A simple and interactive Stopwatch application built with HTML, CSS, and JavaScript.

## ✨ Features

* Start timer
* Pause timer
* Stop and reset timer
* Lap time display
* Clean and simple UI
* Responsive layout
* Custom personal profile section with LinkedIn link

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🎮 How It Works

* Click **Start** to run the stopwatch.
* Click **Pause** to pause the timer.
* Click **Stop** to reset everything.
* Click **Lap** to display the current recorded time.
